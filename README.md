# benjerry

## To Run:

### Requirements:
    Docker >= 17.06ce (Because it uses multi stage build)
    docker-compose(Should be bundled in docker)

```bash
    ./build.sh
```

App will run on port 3000.

If you want to edit the port or any cfg, change it in `docker-compose.yaml`
